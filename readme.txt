Allows WooCommerce customers to upload videos related to their orders and view these videos in the order admin panel.

== Description ==

    Order Return & Refund Policy allows users to upload videos related to their WooCommerce orders. Customers can submit videos directly from their order details page, and admins can view these videos in the order admin panel.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/Order Return & Refund Policy` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Visit WooCommerce > Orders, and you'll see the video upload feature on the order details page for customers.

== Frequently Asked Questions ==

= Can customers upload videos to all orders? =
Yes, this feature is available on the WooCommerce order details page for any completed orders.

= What video formats are supported? =
Currently, MP4/youtube format is supported.
